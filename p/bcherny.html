前 Meta 5 年首席工程师（Principal Engineer），负责 Facebook 和 Instagram 内部开发者工具。《Programming TypeScript》（O'Reilly，2019）作者，是 TypeScript 社区的标准教材之一。加入 Anthropic 后构思了 Claude Code 的最初原型，最早只是一个让 Claude 知道工程师在听什么音乐的 CLI 玩具，后来给它访问文件系统权限，在公司内部迅速流行。2025 年 Claude Code 从 research preview 转为 GA。被业内视为 AI native 工程师的代表实践者（传闻每周写 100+ PR）。被 Pragmatic Engineer、Lenny 等头部技术媒体频繁深度访谈，影响了大批工程师对 agentic coding 的认知。
