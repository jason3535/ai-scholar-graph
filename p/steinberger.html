奥地利软件开发者，2011 年创办 PSPDFKit——一款被 iOS 与企业应用广泛采用的 PDF/文档 SDK，将其做大后逐步退出日常管理。沉寂一段时间后，他以 AI 辅助软件开发的活跃声音重新出现，持续构建并分享对 Agent 式编程工具（Cursor、Claude Code、Codex）的实验，并开源多个 AI 工作流工具。他是「编程 Agent 如何改变软件工程手艺」这一话题被广泛关注的评论者。
Austrian software developer best known for founding PSPDFKit in 2011, a widely used PDF/document SDK adopted across iOS and enterprise apps, which he scaled before stepping back from day-to-day leadership. After a hiatus he re-emerged as a prominent independent voice on AI-assisted software development, building and sharing experiments with agentic coding tools such as Cursor, Claude Code and Codex, and shipping open-source utilities for AI workflows. A widely followed commentator on how coding agents change the craft of software engineering.
